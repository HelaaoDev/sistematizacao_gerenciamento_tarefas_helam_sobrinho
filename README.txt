Protótipo: Sistema de Gerenciamento de Tarefas (versão protótipo)
Arquivos:
- index.html
- styles.css
- app.js

Como usar:
1. Abra o arquivo index.html em um navegador moderno (Chrome, Firefox).
2. Adicione tarefas com título, descrição, prioridade e data de vencimento.
3. As tarefas são armazenadas localmente no localStorage do navegador.
4. É possível editar, concluir e excluir tarefas. Também filtrar e pesquisar.
5. É possível mover os cards de acordo com a necessidade.
6. É possível adicionar "subtasks" caso necessário para maior controle das tarefas.

Observações:
- Este protótipo é uma aplicação client-side (sem servidor).
- Pode ser publicado como Progressive Web App ou adaptado para back-end posteriormente.
